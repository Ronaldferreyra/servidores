Tipos de Servidores Web

Esta es mi investigación sobre servidores

Apache Server

¿Qué es? Es el servidor más usado y antiguo del mercado.

Funciona con pequeños "módulos" que puedes activar o desactivar según lo que necesite tu página web.

Ventaja: Es muy fácil de configurar con archivos .htaccess.

Nginx

¿Qué es? Es un servidor enfocado en la velocidad.

Se usa mucho cuando miles de personas entran al mismo tiempo a una página porque no traba la computadora.

Ventaja: Excelente para imágenes y contenido que no cambia.

Microsoft IIS

¿Qué es? Es el servidor que viene para Windows.

Viene integrado con el sistema operativo y se usa mucho para páginas hechas con lenguajes de Microsoft como .NET.

Ventaja: Tiene botones y ventanas para mover todo con el mouse.

LiteSpeed

¿Qué es? Un servidor premium súper rápido.

Es como el hermano menor pero más rápido de Apache. Hace que páginas como WordPress carguen volando.

Ventaja: No tienes que cambiar nada de Apache para usarlo.

Node.js

¿Qué es? No es un servidor tal cual, pero sirve para hacerlos.

Se usa para aplicaciones modernas en tiempo real como chats o juegos donde todo tiene que actualizarse al instante.

Ventaja: Usas el mismo lenguaje (JavaScript) afuera y adentro.
